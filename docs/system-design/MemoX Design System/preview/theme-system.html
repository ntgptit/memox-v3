MemoX Design System

Tokyo Pure Light · Tokyo Nebula

Calm, mobile-first flashcard app. Plus Jakarta Sans, vibrant indigo primary, deep navy night sky, indigo-tinted outlines. Material 3 tonal surfaces. No emoji, no hype, no gradient chrome.
