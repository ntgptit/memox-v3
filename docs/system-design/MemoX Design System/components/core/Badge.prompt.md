Small status or label pill — use for note states, counts, plan tiers. Soft-tinted by default; pass `solid` for filled.

```jsx
<Badge tone="accent">Pro</Badge>
<Badge tone="success" icon="check">Saved</Badge>
<Badge tone="warn">Draft</Badge>
<Badge tone="danger" solid>Overdue</Badge>
```

Tones: `neutral`, `accent`, `success`, `warn`, `danger`, `info`.
