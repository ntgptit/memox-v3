# System Design

This folder contains product design references used by MemoX agents and developers.

## Contents

| Folder | Purpose |
| --- | --- |
| `MemoX Design System/` | Visual design system, tokens, brand assets, preview pages, and mobile mock design UI kit. |

## Agent rule

Use this folder as design reference only. Production Flutter code must map visual intent to existing `Mx*` shared widgets, Material 3 theme, ARB localization, and Flutter token classes. Do not copy raw HTML inline styles, raw hex values, emoji placeholders, or hardcoded layout values into production widgets.
